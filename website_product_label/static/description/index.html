Odoo Website Product label & Stickers
Eye Catching Labels & Stickers are ready to help you in Increasing Sales

Features

  • Provide Customizable Set of Images Label Packs.
  • Show Eye Catching Labels In Product View.
  • Labels will also visible on single product view.
  • Preview Option To judge Best Position.
  • Admin Can Set Label Position like
    • Place on Left, Left-Top, Left-Bottom.
    • Place on Center, Center-Top, Center-Bottom.
    • Place on Right, Center-Top, Right-Bottom.
    • Also Adjust Label position in Pixels.

Work-flow

Labels & Stickers

After the successful installation module you will be able to see the Product Label menu under Website -->Products --> Configuration.

Assign Label To Product

Manually Select the label in Website -->Products-->>Sales.

Have A Preview

Admin can preview that how the label will actually look before the final assignment.

Customize Label Properties

Odoo Labels & Stickers Module also provide the functionality to customize the label.

Admin can set the Position, Height, Weight of Labels & Stickers.
